Trouble Shooting

- UITableCell안에 UIScrollView를 통해 게시물에 올려진 여러개의 사진을 보기 위해 기능을 제현하였으나
  UICollectionView로 활용하는게 데이터 사용이나 가시성에 효율적이라는 결론을 내어 새로 수정.
